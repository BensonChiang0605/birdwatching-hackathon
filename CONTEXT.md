# Birdwatching

This context describes the shared language for turning a photographed bird encounter into a social, collectible observation.

## Language

**Sighting**:
A single photographed bird encounter contributed by a person. It may include a source photo, bird cutout, suggested species, optional nickname, observed place, and observed time.
_Avoid_: Bird page, post, activity

**Species**:
The taxonomic kind of bird suggested by the identification model for a Sighting. It does not identify an individual animal.
_Avoid_: Bird identity, breed

**Nickname**:
An optional human-given display name for the bird shown in one Sighting. It does not imply that the same individual can be recognized in another Sighting.
_Avoid_: Species name, identity

**Community Feed**:
The chronological collection of Sightings shared by contributors.
_Avoid_: Home page, activity database
